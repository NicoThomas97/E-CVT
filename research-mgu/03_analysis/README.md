# 03_analysis 分类入口

`03_analysis` 现在按研究流程拆成子目录。这里只放分类说明和入口，不再继续堆平铺文件。

## 目录说明

- `00_entry_and_index/`：研究重点、竞品网页索引。
- `10_market_and_scenarios/`：市场、痛点、场景、替代方案、OEM 架构选择。
- `20_architecture_and_benchmarks/`：技术架构、竞品深挖、历史启发、范围基准。
- `30_product_definition/`：产品定义、控制逻辑、展示故事板。
- `40_feasibility_and_validation/`：体感可实现性、仿真模型、验证计划、Go/No-Go。
- `90_source_materials/`：原始或半原始材料、网页摘录、Excel、HTML、图片目录。
