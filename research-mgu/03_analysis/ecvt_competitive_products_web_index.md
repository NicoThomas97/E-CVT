# e-CVT MGU 竞争/替代产品与网页索引

生成日期：2026-07-07

## 1. 检索口径

本文件保存当前调研中可与 e-CVT MGU 形成竞争或替代关系的产品/系统，并按网页类型整理官网功能介绍、展会/发布报道、OEM 装车页和骑行体验/测评页。

纳入范围：

- 直接竞争：电机与变速机构集成的 MGU、e-CVT MGU、motor + CVT / gearbox 一体化方案。
- 强替代：外部电子/自动换挡、花鼓 CVT/内变、齿轮箱、皮带低维护传动等会抢占 e-CVT 价值点的方案。
- 边界线索：历史项目、原型、Bosch eShift 伙伴系统、配套生态。此类不能直接当作活跃竞品结论。

排除范围：

- 单纯普通中置电机性能基准，例如 Avinox M2S/M2/M1、Bosch Performance Line CX，除非页面用于解释竞争背景。
- 泛泛品牌新闻、无具体产品/系统信息的页面。
- 未能对应到功能、展会/发布、骑行体验、OEM 装车或明确技术边界的页面。

## 4. 高优先级跟踪对象

优先级最高：

- Avinox MG Concept：补官方 press kit、最终规格表、量产时间、内部机制、OEM prototype 页面。
- Gobao X-system / X1 / X1P：BikeBiz 已补 X1/X1P、2027 年 2 月量产和 OEM 服务策略线索；仍需 Gobao 官方命名页、OEM 客户、效率/热/寿命闭环。
- Pinion E-Drive MGU E1.9/E1.12：补更多长期骑行体验、噪声、故障、维修成本、OEM 价格带。
- Owuru / E2Drives：官方城市车技术链已补齐；继续追高功率 datasheet、Rockrider/Decathlon 量产计划、非 Decathlon OEM 合作、效率/热/寿命和长测。

强替代持续跟踪：

- Bosch eShift、Shimano AUTO SHIFT、SRAM Eagle Powertrain：重点看 eMTB / trekking / cargo 中外部自动换挡能否足够好，是否削弱 e-CVT MGU 的必要性。
- enviolo AUTOMATiQ、Rohloff E-14、3X3 Electric Shift：重点看 cargo、urban、SUV/trekking 场景中低维护/自动/静止换挡的替代压力。
- Gates Carbon Drive：作为低维护传动配套，关注与 MGU/gearbox 的车架接口、张紧、寿命和售后约束。

## 5. 证据边界

- Avinox MG Concept 当前仍以媒体、行业报道和展台图卡为主；不能把 Avinox M2S/M2/M1 的功率、重量、扭矩直接套用到 MG Concept。
- Gobao 官方 X-system 页面支持 e-CVT MGU 方向；X1/X1P 命名和 2027 年 2 月量产节奏已有 BikeBiz 发布页核验，但仍需 Gobao 官方规格/命名页面闭环。
- Pinion E-Drive MGU 是当前最清楚的量产 MGU 参照，但它是有级 gearbox MGU，不是 e-CVT。
- Bosch/Shimano/SRAM/enviolo/Rohloff/3X3 不是电机内部 e-CVT，但会抢占自动换挡、低维护、静止/负载换挡和用户不用操心档位这些核心价值点。
- Valeo、Revonte 当前只作为历史线索，不能列为活跃竞品。
- Decathlon LD 920 E / Owuru 证明城市车目标踏频 UX 可成立，但不能直接证明高功率 eMTB/cargo e-CVT MGU 的热、寿命、效率和可靠性。
