## 3. 五层完整正文

### 第 01 层：e-CVT 是什么

在本调研口径中，e-CVT MGU 指中置电机与连续变比机构集成为单一驱动单元，通过目标踏频、扭矩平滑、连续变比和虚拟档位降低用户换挡负担，并把单速链条/皮带外传动和整车系统功能纳入同一架构。

- 它不是“普通中置电机 + 自动换挡”的同义词，而是把电机、传动、控制和整车诊断放进同一系统边界。
- Gobao X-system 是当前最强官方证据：页面支持电机与传动集成在单一单元、自动连续变比、目标踏频和联网服务/安全功能。
- Avinox MG Concept 已升级为量产方向/准量产目标参照：技术细节图卡可用于内部对标，但最终规格表、内部机械实现和测试基准仍需闭环。

证据：`V004`、`V032`、`F026`、`F027`、`F043`、`F048`

### 第 02 层：e-CVT 可以用到哪些场景

e-CVT MGU 理论上可以进入多种高端电助力自行车场景；Avinox 技术细节图卡明确覆盖 eMTB、电助力长途、多用途、eGravel、城市和载货，但可用不等于适合。

- 高端 eMTB：后拨/飞轮/链条外露、非簧载质量、扭矩平滑、负载换挡和技术爬坡是强痛点，但重量、效率、NVH 和骑手信任门槛最高。
- 载货 / 家庭工具车：满载起步、坡道、频繁停启、低技能用户误操作、低维护/全生命周期使用成本、倒车助力和机械锁止是核心痛点。
- 高端多用途车 / 长途通勤：低维护、目标踏频、长距离混合路况、轻越野、载物、内置速度传感和整车系统集成可以形成综合价值。

证据：`V004`、`V032`、`F026`、`F041`、`F044`、`F048`

### 第 03 层：已有方案具体怎么解决

现有方案已经用不同方式解决了自动换挡、低维护、宽范围、封闭传动和量产风险。第三层只讲它们怎么解决问题，不急着评价 e-CVT。

- 电子外变 + 中置：Bosch、Shimano、SRAM 用传感、电机卸力、自动换挡、滑行换挡、手动覆盖、应用/显示生态解决换挡时机和运动用户熟悉度。
- 花鼓 CVT/内变 + 中置：enviolo、Rohloff、3X3 用目标踏频、自动无级、起步档、静止/负载换挡、封闭低维护解决停启、省心和维护问题。
- 有级齿轮箱 MGU 与普通中置：Pinion 用封闭齿轮箱、宽范围和服务叙事解决外露传动；普通中置用成熟、轻、便宜和维修网络解决量产风险。

证据：`V001`、`V005`、`V007`、`V008`、`V009`、`V010`、`V012`、`F041`

### 第 04 层：剩余问题与 e-CVT 补法

第四层才进入 e-CVT：逐项回答已有方案还留下什么问题，e-CVT 如何用中置集成、连续变比、单速外传动和系统功能补位。

- 电子外变剩后拨/飞轮/链条外露风险；e-CVT 用内部变比 + 单速链条/皮带外传动减少后端易损件，并形成非簧载质量降低叙事。
- 花鼓 CVT/内变剩后轮复杂、轮组维修和诊断分散；e-CVT 把复杂机构放回车架中部，提升后轮简化、悬架/货架布置和整车诊断收益。
- 有级 MGU 和普通中置仍有踏频自然感或整车功能不足；e-CVT 用连续变比、虚拟档、扭矩平滑、机械锁止、倒车助力和内置速度传感补位。

证据：`V032`、`F018`、`F020`、`F026`、`F027`、`F044`、`F047`

### 第 05 层：e-CVT 真正适合的场景

真正适合的场景，是替代方案难以同时做到低维护、目标踏频、后端简化、低速高扭矩和整车系统收益的场景。

- 高端 eMTB 是强痛点、高战略价值、高验证门槛场景，Avinox 已补强非簧载质量、扭矩平滑、负载换挡和虚拟档位价值点。
- 载货 / 家庭工具车是强痛点场景，但花鼓 CVT/内变替代压力强，必须用高载热、寿命和全生命周期使用成本数据证明。
- 高端多用途车 / 长途通勤是综合价值最均衡的场景，最容易把低维护、目标踏频和高端一体化驱动体验讲清楚。

证据：`V004`、`V009`、`V020`、`V022`、`V032`、`F026`、`F027`、`F041`、`F044`、`F048`

## 4. 适合场景卡片正文

- 高端 eMTB / 重力骑行 / 技术爬坡：eMTB 的后拨、飞轮、链条外露易损，泥沙、石击、摔车、跳跃落地和低速高扭矩会放大传动风险。e-CVT MGU 的潜力在于后端简化、非簧载质量降低、扭矩平滑、技术爬坡和旗舰差异化。
- 载货 / 家庭工具车 / 重载：满载起步、坡道、频繁停启、家庭用户误操作和低维护/全生命周期使用成本是强痛点。e-CVT 的机会在于让载货/载娃场景像自动挡车一样易用，但花鼓 CVT/内变是非常强的替代路线。
- 高端多用途车 / 长途通勤 / 全路况通勤：痛点不一定最极端，但低维护、目标踏频、轻越野、载物、长距离混合路况和整车联网/安全功能更容易组合成高端一体化驱动体验。
- 高端城市通勤：目标踏频和不用操心换挡的用户体验价值成立，但 enviolo/Owuru/花鼓路线的替代压力很大。
- 长途旅行 / 探险：长途低维护需求真实，但 Rohloff、Pinion 和齿轮箱路线的可靠性叙事强。
- 轻量 eMTB / eGravel：轻、自然、低阻和运动控制优先，e-CVT 的重量、效率和体积成本很难被价值覆盖。
- 车队 / 配送 / 共享工具车：潜在全生命周期使用成本场景，但必须量化维护频次、停工时间、远程诊断和模块更换效率。

## 6. 证据 ID 和证据边界

- `V004` / `F026` / `F027`：Gobao X-system 是当前最强官方 e-CVT MGU 证据，但仍不能外推未确认 OEM 客户、真实量产状态、长期可靠性和效率图谱。
- `V032` / `F043` / `F044` / `F045` / `F046` / `F047` / `F048`：Avinox MG Concept 技术细节图卡可作为量产方向/准量产目标参照，但仍是展台图卡/OCR，不是最终公开规格表；内部机制、测试基准和认证仍需闭环。
- `V001` / `F001` / `F004`：Pinion 证明有级齿轮箱 MGU 的量产参照和服务叙事，但不是 e-CVT。
- `V005` / `V007` / `V008` / `F013` / `F016` / `F018`：Bosch、Shimano、SRAM 证明电子外变和自动换挡生态的替代压力，不是电机内部变速证据。
- `V009` / `V010` / `V011` / `V012` / `F020` / `F024` / `F041`：enviolo、Rohloff、3X3 证明花鼓 CVT/内变可以解决许多城市、长途和载货场景痛点，因此也是强替代路线。
- `V020` / `V022`：Simplon KAGU :ePinion High 证明高端多用途/长途整车中已有 MGU 参照，但媒体体验不能替代实验室测试和市场表现数据。
